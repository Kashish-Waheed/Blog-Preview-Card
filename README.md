A responsive Blog Preview Card built with HTML and CSS.

This is a solution to the Blog Preview Card challenge from Frontend Mentor. The goal of this project was to recreate the design as close as possible using HTML and CSS.
Frontend Mentor challenges helps improve coding skills by building real-world projects.

Responsive layout
